# Tabs

> Show content in a tabbed interface.

```jsx title="Import"
import { Tabs } from "@webeetle/windy";
```

## Base

```jsx live
<div className="w-full">
  <Tabs>
    <Tabs.Tab label="Tab 1" />
    <Tabs.Tab label="Tab 2" />
    <Tabs.Tab label="Tab 3" />
    <Tabs.Tab label="Tab 4" />
  </Tabs>
</div>
```

## Color

```jsx live
<div className="w-full">
  <Tabs color="yellow">
    <Tabs.Tab label="Tab 1" />
    <Tabs.Tab label="Tab 2" />
    <Tabs.Tab label="Tab 3" />
    <Tabs.Tab label="Tab 4" />
  </Tabs>
</div>
```

## Current Index

```jsx live
() => {
  const [current, setCurrent] = React.useState(0);
  return (
    <div className="w-full">
      <Tabs onChange={({ selectedTab }) => setCurrent(selectedTab)}>
        <Tabs.Tab label="Tab 1">Tab 1</Tabs.Tab>
        <Tabs.Tab label="Tab 2" />
      </Tabs>
      <div className="pt-10 text-center">
        Hello from Windy {current}
      </div>
    </div>
  );
}
```
